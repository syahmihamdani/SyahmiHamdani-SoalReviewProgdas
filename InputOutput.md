# Input Output

## Soal

Buatlah sebuah program yang menerima 2 angka A dan B, dan mencetak hasil pembagian dari A/B dengan pendekatan **2 angka dibelakang koma**.

## Test Case 1

Input

```
5 2
```

Output

```
2.50
```

## Test Case 2

Input

```
1 3
```

Output

```
0.33
```

## Test Case 3

Input

```
67 16
```

Output

```
4.19
```
