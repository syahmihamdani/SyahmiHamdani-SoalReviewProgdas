# Function

## Soal

Program akan menerima input 2 buah angka, yakni X dan Y. Cetak hasil dari fungsi f(x,y) sebagai berikut

$$
f(x,y) = x^2y^2 + \frac{x + y}{xy}
$$

Cetak hasil dengan pendekatan **2 angka dibelakang koma**.

## Test Case 1

Input

```
12 7
```

Output

```
7056.23
```

## Test Case 2

Input

```
3 5
```

Output

```
225.53
```

## Test Case 3

Input

```
5 10
```

Output

```
2500.30
```
