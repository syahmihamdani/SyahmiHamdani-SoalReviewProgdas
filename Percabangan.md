# Percabangan

## Soal

Program akan menerima input berupa angka dari rentang 0-100, dan tampilkan nilai huruf sesuai dengan tabel di bawah:

Nilai    | Huruf
---------|-------
85 - 100 | A
80 - 84  | A-
75 - 79  | B+
70 - 74  | B
65 - 69  | B-
60 - 64  | C+
55 - 59  | C
50 - 54  | C-
40 - 49  | D
0 - 39   | E

## Test Case 1

Input

```
90
```

Output

```
A
```

## Test Case 2

Input

```
60
```

Output

```
C+
```

## Test Case 3

Input

```
78
```

Output

```
B+
```
